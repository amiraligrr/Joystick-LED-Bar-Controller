### pins :
* joyStick X to :A1
* led 1 to: 8 
* led 2 to: 2
* led 3 to: 3 
* led 4 to: 4 
* led 5 to: 5 
* led 6 to: 6 
* led 7 to: 7 
* joy stick key to :A3
